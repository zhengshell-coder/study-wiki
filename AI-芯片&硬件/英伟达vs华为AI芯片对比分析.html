执行摘要

两家走的不是同一条路。英伟达靠最先进制程 + CUDA 生态打"单芯片性能 + 通用生态"的硬仗； 华为在被切断先进制程后，转向"系统级架构 + 自研内存 + 全栈自主"的打法，用超节点（SuperPoD）把成千上万颗中等性能的芯片用全光互联拼成一个超大算力域，以系统赢回单芯片输掉的部分。 单芯片层面英伟达仍领先约一代到两代；系统/集群层面与价格层面华为已具备正面竞争力。中国市场正快速向昇腾倾斜，全球市场则仍是英伟达 + CUDA 的天下。

代差到底意味着什么:单卡差距如何被部分抵消

单卡看,B200 的 FP4 算力约为昇腾 950PR 的 4–5 倍,代差明显。但终端有效差距远没有这么大,因为有三重抵消:

  • 系统级抵消:华为用超节点(全光 UB)把更多中等卡拼成一个大算力域,集群层面差距收窄——CloudMatrix 384 的系统算力(稠密 BF16)约为 NVL72 的 1.7–2 倍。
  • 算法/软件抵消:模型侧效率优化(MoE、低精度、KV-Cache、并行策略,如 DeepSeek 的训练/推理优化)降低了对单卡峰值的依赖。
  • 场景抵消:推理场景对单卡峰值要求低、更看重显存/带宽/成本,差距进一步缩小(详见市场章)。
可以这样理解:有效差距 ≈ 代差 ÷(系统规模 × 算法效率 × 场景适配)。但分母放大并非无限——堆系统要付出更多卡、功耗、占地与成本,且算法红利两家都能吃;在大模型训练、单卡密集算力上,制程代差仍是绕不开的硬约束。

②两家芯片研究方向的异同（拆开来讲）

这是理解两者差异的关键——它们在解一道同样的题（怎么提供更多有效 AI 算力），但约束条件不同，于是给出了截然不同的工程哲学。先分开讲各自方向，再讲异同。

🛣️ 两条技术路线示意
🟢 英伟达 · 芯片优先(Compute-first)
💪 强单芯片
最先进制程 → 🔗 铜缆 NVLink → 🗄️ 单柜 = 超节点 → 🌐 IB 横向扩集群
🔴 华为 · 系统优先(System-first)
🧩 中等芯片 ×N
制程受限 → 💡 全光 UB 总线 → 🏗️ 超大算力域 → 🌐 RDMA 扩集群

同样的目标(更多有效算力),不同的出发点:英伟达从"单芯片最强"起步,华为从"系统最大"起步。

🟢 英伟达：芯片优先 (Compute-First)

  • 极致单芯片：用最先进制程（4nm→3nm）、最大裸片、最高密度晶体管堆出单卡峰值算力。
  • 纵向 + 横向扩展：机柜级 scale-up（NVL72）+ 集群级 scale-out（InfiniBand/Spectrum-X）。
  • 生态护城河：CUDA + 库 + 框架 + 网络（收购 Mellanox），软硬一体绑定开发者。
  • 推理成本曲线：Rubin 主打把单 token 推理成本相对 Blackwell 降一个数量级。
  • 精度演进：持续下探低精度（FP8→FP4），换取吞吐与能效。

🔴 华为：系统优先 (System-First)

  • 系统补芯片：单芯片落后就用"超节点"——把数千卡组成一个扁平、低延迟的大通信域，整体算力反超。
  • 全光互联：发挥三十年通信积累，用光模块替代铜缆，突破距离/带宽限制（卡间百纳秒级）。
  • 自研内存：HBM 被卡，就自研 HiBL/HiZQ，绕开海力士/美光供应。
  • 全栈自主可控：制程、内存、互联、CANN 软件栈尽量国产闭环，抗断供优先于追平制程。
  • 场景拆分：950PR 专攻推理 Prefill/推荐，950DT 专攻 Decode/训练——按负载分化产品。

相同之处

  • 都在押注低精度计算：FP8 / FP4(MXFP4) 成为两家算力标称的主战场，反映推理成本是共同的优化目标。
  • 都从"卖芯片"转向"卖系统/机柜":NVL72 与 Atlas/CloudMatrix 超节点本质都是把互联和整柜方案作为核心产品。
  • 都重视互联带宽：都认为"内存墙 + 通信墙"是大模型时代真正的瓶颈，争相提升卡间带宽。
  • 都在构建软件锁定：CUDA 与 CANN+MindSpore 都想成为开发者的默认栈。

不同之处（核心分歧）

分歧点英伟达华为
取胜单元单颗 GPU 的绝对性能整个超节点/集群的系统性能
互联介质铜缆 NVLink（成本低、距离短）全光互联（成本高、距离长、带宽大）
内存来源外购 HBM3e/HBM4自研 HBM（被迫，但换来自主）
制程依赖强依赖台积电领先节点主动降低对先进制程的依赖
战略动因商业竞争、维持技术代差制裁倒逼下的"用工程绕过封锁"
能耗取向不惜高功耗换峰值更强调单位能效与系统散热友好
本质：华为的全光超节点路线，很大程度上是美国制裁的直接产物——既然拿不到最先进制程和外购 HBM，就把战场从"单芯片"挪到自己最擅长的"通信与系统集成"上。这是约束催生的差异化，而非单纯的技术偏好。
对行业的影响:路线分化正外溢到整条产业链——英伟达路线拉动先进制程、HBM、铜缆 NVLink、InfiniBand;华为路线拉动光模块、光器件、液冷、国产 HBM 与先进封装。对用户而言,选英伟达是"性能强 + 生态成熟,但受供应与价格约束",选华为是"高性价比 + 自主可控,但要承担生态迁移与系统复杂度"。两条路线之争,本质也是两套供应链与两类受益方之争。

③昇腾 950 系列目前的短板

950 系列在系统、价格、自主可控上亮点突出，但仍有结构性短板，主要集中在制程、单芯片性能、生态和制造端。其短板本质是一条因果链:

⛓️ 短板因果链:一个根因,层层传导
🚫 制程受限
无 EUV / 台积电 → 📉 单芯片弱
算力/带宽落后 → 🧱 靠堆系统补
超节点万卡 → ⚠️ 三重压力
产能 · 成本 · 生态
根因(制程)传导环节最终代价

短板1制程落后，存在代际差距

无法使用台积电先进节点，受限于国产制程（约 7nm 级、无 EUV），相对 B200 的 4nm 落后约 1–2 代。这直接压制单芯片性能上限与能效天花板，并需要更多芯片才能堆出同等算力。

短板2单芯片算力与带宽仍明显落后

950PR FP8 仅 1 PFLOPS、显存带宽 1.6 TB/s，而 B200 FP4 约 9 PFLOPS、带宽 8 TB/s。单卡密集训练/复杂推理场景仍处于劣势，必须靠超节点规模来弥补。

短板3软件生态与 CUDA 仍有差距

CANN/MindSpore 工具链、算子库、社区成熟度、第三方框架适配度都不及 CUDA 二十年的积累。虽宣称兼容 90%+ CUDA 代码一键迁移，但真实迁移中性能调优、长尾算子、边角兼容仍是迁移摩擦点，开发者习惯迁移需要时间。

短板4自研 HBM 处于早期

HiBL/HiZQ 是首发自研内存，容量/带宽（128–144GB、1.6–4 TB/s）虽追平到 HBM3 级别，但相比 HBM3e/HBM4 仍有差距，且良率、成本、量产稳定性尚待长期验证。

短板5产能与良率约束（最关键的现实瓶颈）

国产先进制程产能有限、良率虽达 80%+ 但产线扩张受设备（光刻机）管制制约。"系统级堆卡"的打法对芯片数量需求极大（一个超节点动辄数千上万卡），这反过来对本就紧张的产能形成巨大压力——能否大规模供货，是比纸面参数更现实的短板。

短板6系统功耗与 TCO 的隐性代价

单芯片弱→需要更多芯片→单芯片能效虽好，但整系统的占地、光模块成本、整体电耗和运维复杂度上升。"用数量换性能"在超大规模下会推高整体拥有成本与数据中心基础设施压力。

归纳：950 系列的短板可压缩为一句话——"制程受限 → 单芯片弱 → 靠堆系统补 → 转嫁为产能、成本、生态三重压力"。其中产能/良率是最现实的天花板，生态成熟度是最持久的追赶项。

推理是国产替代的突破口:对单卡峰值要求低,更看重显存/带宽/成本/吞吐与能效,国产卡(昇腾、寒武纪、昆仑芯、PPU)在推理已接近或部分超过 H20。训练仍是英伟达腹地:大模型训练需要单卡密集算力 + 成熟分布式训练栈(NCCL/并行框架)+ 生态,国产追赶更难。随着模型进入大规模落地期,推理算力需求增速快于训练——需求结构向推理倾斜,正好利好国产替代。替代路径因此大概率是"先推理、后训练":谁先在推理规模化站稳,再逐步向训练渗透。

全球市场：英伟达 + CUDA 的护城河仍稳固

全球范围内，英伟达凭借 CUDA 生态、最先进制程、机柜级方案和先发的开发者锁定，仍占据绝对主导（数据中心 AI 加速器份额长期 80%+），B200/GB200 已被预订到 2026 年中、积压订单数百万颗，Rubin 接力。华为受出口与生态限制，海外市场拓展空间有限，短期内昇腾主要是"中国及部分友好市场"的故事，而非全球替代。

真正可能蚕食英伟达全球份额的，更可能是云厂商自研 ASIC（谷歌 TPU、亚马逊 Trainium、微软 Maia 等）和 AMD MI 系列，而非华为。

🌐 最可能的格局

"一球两制"——全球市场英伟达 + CUDA + 云自研 ASIC 主导；中国市场昇腾领衔、多家国产并存。两个生态逐步分叉，软件栈、互联标准、精度格式各自演进。

⚠️ 关键变量

出口管制松紧、国产制程/HBM 产能爬坡、CANN 生态成熟速度、以及英伟达能否拿到合规重返中国的产品——任一变化都可能改写份额曲线。

⑤决定格局的核心因素

🏛️

1. 地缘政治与出口管制

最大单一变量。管制收紧 → 加速国产替代；松动 → 英伟达合规产品回流。中国市场格局基本由政策曲线决定。

🏭

2. 先进制程与产能

EUV 光刻机管制、中芯国际良率与扩产速度，决定昇腾能"堆"多少卡。产能是比参数更硬的约束。

💾

3. 高带宽内存 (HBM)

大模型的真正瓶颈在内存墙。自研 HBM 的容量/带宽/良率追赶速度，直接决定昇腾天花板。

🔗

4. 互联与系统架构

全光 vs 铜缆、scale-up vs scale-out。谁能把更多芯片低延迟地连成一个域，谁就能在系统层赢回单芯片的差距。

🧩

5. 软件生态与迁移成本

CUDA 的二十年锁定 vs CANN 的兼容追赶。开发者习惯、算子库、框架适配是最持久的护城河，也是最慢的变量。

💰

6. 成本 / TCO 与性价比

昇腾单卡约对标产品 1/3 价格，但要算上系统、光模块、电耗的整体 TCO。性价比是国产替代的核心说服力。

🤖

7. 下游需求结构（训练 vs 推理）

推理对单芯片极致性能要求较低、对成本敏感，是国产芯片最先攻破的市场；训练仍是英伟达腹地。需求向推理倾斜利好华为。

☁️

8. 自研 ASIC 与客户集中度

大客户（云厂商）自研芯片是全球层面侵蚀通用 GPU 份额的关键力量，也压缩英伟达和华为的通用算力空间。

⑥兆瓦 / 吉瓦级数据中心造价测算

本章测算一个给定供电规模（IT 负载）的 AI 数据中心的总建造成本——分别按"美国 + 英伟达方案"与"中国 + 华为方案"两种情形，其余条件尽量一致。注意：这是基于业界公开成本结构的量级测算，非工程报价；芯片/电价/土建价格波动极大，结论看量级与结构，不看小数点。

测算逻辑（业界标准成本结构）

AI 数据中心的总投资（CapEx）通常按"每瓦 IT 负载"为单位测算，再乘以规模。业界把成本拆为两大块：

  • IT 设备（服务器 / GPU / 网络）≈ 全部 CapEx 的 60–64%——这是 AI 数据中心与传统机房最大的不同，加速卡是最大单项。
  • 物理基础设施 ≈ 35–40%，其内部再拆为:供配电 40–45%、冷却 15–20%、土建外壳 + 土地为余下部分。

行业基准:仅土建外壳约 $10–12M / MW;含 GPU 满配的"全包 AI 建设"约 $30–40M / MW;一个 1 GW 级 AI 数据中心前期 CapEx 约 $38B(服务器约占 60%)。下面以此为锚点,套用中美差异系数。

核心变量(两国差异):① 芯片单价与每卡性能;② 电力/供配电设备成本;③ 土建与土地;④ 人工。美国方案贵在芯片单价 + 高功率配套;中国方案省在土建/电力/人工,但同等供电下需更多张卡且单卡算力较低。

🔴 寒武纪

国产 · A股算力龙头

思元(MLU)系列,字节等采购,国产替代核心标的之一,推理场景性价比突出。

🔴 阿里平头哥 / 百度昆仑芯

国产 · 自研自用

阿里 PPU(2025 市占约 6.6%)、百度昆仑芯 P800,均走"自研芯片 + 自家模型/云"闭环路线。

格局含义:全球市场是"英伟达 + AMD + 云自研 ASIC"的竞争;中国市场是"昇腾领衔 + 寒武纪/平头哥/昆仑芯/海光并存"。华为的主战场是中国及部分友好市场,真正威胁英伟达全球地位的,是云厂商的自研芯片。
更深一层的逻辑:为什么"第三极"里云自研 ASIC 才是英伟达全球地位的真正变量?因为大客户(谷歌/亚马逊/微软/Meta)既是英伟达最大买家、又在自研——需求一旦"内化",就直接从英伟达订单里扣除;且自研芯片针对自家负载优化、性价比更高。但 ASIC 通用性与生态不及英伟达,所以是"分流"而非"取代"。中国国产则是政策 + 成本驱动的确定性替代:确定性高,但厂商分散、生态各自为政,是其相对华为/英伟达的短板。

🎯 结论与展望

把全篇线索收拢为几条核心判断,并给出两种政策情景下的走向。

五条核心判断

  1. 单芯片与生态:英伟达短期难撼动。制程领先 1–2 代 + CUDA 二十年护城河,全球主导地位短期稳固。
  2. 中国市场:政策驱动,快速向昇腾倾斜。"一球两制"的分叉格局正在成型,两个生态各自演进。
  3. 系统级路线有效但有顶。华为超节点能在集群层面正面竞争,但受国产制程产能/良率天花板制约——这是比纸面参数更现实的瓶颈。
  4. 全球真正的变量是云自研 ASIC,而非华为。谷歌 TPU、AWS Trainium 等更可能蚕食英伟达全球份额。
  5. 决胜点已转移。从"单芯片最强"转向"每美元 / 每瓦有效算力 + 生态成熟度",并日益受电力与产能约束。

两种政策情景

🔒 管制收紧

国产替代进一步加速,昇腾及国产份额快速上行,中美生态分叉加深;华为产能/良率与生态成熟成为决定上限的关键。

🔓 管制放松

英伟达合规产品回流中国,昇腾短期承压;但国产已建立根据地与供应链,替代趋势不会逆转,只是节奏放缓。

一句话收尾:这不再是"谁的芯片更快"的单点比拼,而是两套技术哲学、两个软件生态、两条供应链在不同约束下的长期博弈。英伟达赢在当下的性能与生态,华为赌的是系统创新、自主可控与时间。
