Court球场标准尺寸 23.77m × 8.23m（单打）；"court" 一词源于中世纪的庭院
Net球网中央高度 0.914m，两端 1.07m；早期网球曾用手拉绳代替

Singles Sideline单打边线界定单打有效区域的两侧纵向线
Doubles Sideline双打边线双打时有效区域更宽，包含双打走廊（alley）
Alley / Tramline双打走廊单打与双打边线之间的狭长区域，英国人称之为 tramline
Tennis Bag球包用于装球拍和装备的专用包袋

🏓 击球技术 / Strokes
English中文词源 / Origin
Forehand正手击球以持拍手同侧挥拍击球；"fore" + "hand" 即"前面的手"
Backhand反手击球以持拍手对侧挥拍击球；分为单手和双手反手
Serve / Service发球每一分的起始击球，将球抛起后击入对方发球区
Volley截击 / 凌空抽球在球落地前直接击回；源自法语 volée（飞翔的一击）
Smash / Overhead高压球 / 扣杀类似于羽毛球扣杀，在头顶上方将球猛力向下击打
Slice切削球球拍从上向下"切"球，产生下旋，使球低弹并减速
Topspin上旋球球拍从下向上刷球，产生上旋，使球弧线更高并快速下落
Drop Shot放小球 / 吊短球轻触球使其刚好过网落地，打对手措手不及
Half Volley半截击 / 半弹球在球刚弹起时立即击球，介于截击和地面击球之间
Approach Shot随球上网击球在底线打出攻击性球后向网前移动的过渡击球
Lob高吊球将球高高打过对手头顶，迫使对手后退；可进攻也可防守
Swing Volley挥拍截击在半高位置用完整挥拍动作打出的攻击性截击
Inside-Out Forehand侧身正手用正手击打反手位来球，球从内侧向外侧飞行
🏐 发球 / Serve
English中文词源 / Origin
First Serve一发每个发球局的第一次发球机会，通常追求速度和角度
Second Serve二发一发失误后的第二次发球，通常更保守以确保成功率
Flat Serve平击发球球几乎无旋转，以速度见长
Slice Serve切削发球带有侧旋，球在落地后向外偏移（从发球者视角）
Kick Serve / Topspin Serve上旋发球 / 踢式发球强烈上旋使球高弹且弹跳方向偏向对手反手侧
AceAce 球（发球直接得分）对手未触球即得分；"ace" 源自扑克牌中的 A，寓意顶尖
Double Fault双误两次发球均失误（下网或出界），直接丢分
Let擦网重发发球擦网后落入正确发球区，判为 let，重新发球；"let" 源自古英语"阻碍"
📊 比赛与计分 / Match & Scoring
< class="cn">平分（40-40）
English中文词源 / Origin
Point分网球计分的最小单位
Game局赢得至少 4 分且领先对手 2 分即赢得一局
Set盘赢得至少 6 局且领先对手 2 局即赢得一盘
Match场比赛通常为三盘两胜或五盘三胜制
Deuce源自法语 deux（二），因须连续赢两分才能获胜
Advantage (Ad)占先deuce 后再赢一分即占先，再赢一分则拿下此局
Love零分可能源自法语 l'œuf（蛋），因蛋形似零；或源于荷兰语 lof（荣誉）
Break Point破发点接发球方再赢一分即可"破发"赢得该局的时刻
Break破发接发球方赢得了发球方的发球局
Hold保发发球方成功守住自己的发球局
Tiebreak抢七6-6 平局时进入抢七，先得 7 分且领先 2 分者赢此盘
Super Tiebreak超级抢七 / 抢十某些赛事在决胜盘使用，先得 10 分且领先 2 分者胜
Match Point赛点再赢一分即赢得整场比赛的关键时刻
Set Point盘点再赢一分即赢得该盘的关键时刻
Triple Bagel三蛋（6-0, 6-0, 6-0）以三个 6-0 的比分获胜，极具统治力的表现

English中文词源 / Origin
Rally对拉 / 多拍回合双方连续击球的回合过程

Baseline Game底线型打法以底线击球为主，不频繁上网的战术风格

Construction of Point分的构建通过连续有计划的击球来创造得分机会的策略思维
Counterpunching防守反击以出色的防守和借力打力来消耗对手的战术风格
